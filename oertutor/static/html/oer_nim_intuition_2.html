This shows several example Nim scenarios. The orange rectangles represent stacks of objects. The number of objects on the stack is shown by the white number in the stack. Each example is combined with the description of the optimal move.

Scenario 1

Take two objects from the third stack, which will leave two identical stacks for the opponent. Whichever one he chooses, you will be able to pick the last one.

Scenario 2

Take one objects from the second stack, which will leave two identical stacks for the opponent. Whichever one he chooses, you will be able to pick the last one.

Scenario 3

Take three objects from any of the stacks, which will leave two identical stacks for the opponent. Whichever one he chooses, you will be able to either pick the last object or to create two identical stacks again.
